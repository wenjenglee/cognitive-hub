# SOP Template — Standard Operating Procedure

> **Purpose:** Step-by-step guide for recurring workflows. SOPs are *how* — mutable procedures that improve over time. Rules are *what and why* — keep them separate.

## How to Use This Template

1. Copy this file when you have a recurring task that benefits from a consistent process.
2. Write steps in the order they happen — someone following this for the first time should be able to complete the task.
3. Include the "Why" for non-obvious steps — this helps AI agents (and future you) understand intent, not just procedure.
4. Review after every 3–5 uses and update based on what actually happens.
