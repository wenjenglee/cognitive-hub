# Project Template — Async Relay Workflow

> **Purpose:** Standard page structure for team projects. Designed so that any team member (or AI agent) can pick up where someone left off without a synchronous meeting.

## How to Use This Template

1. Copy this file when starting a new project.
2. Fill in the header and Goal section first — everything else can grow over time.
3. Share the page with all team members and AI agents involved.
4. Update the checklist and lessons as work progresses.
